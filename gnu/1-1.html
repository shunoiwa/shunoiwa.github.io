1-1. 命令の実行

このセクションでは、Gnuplotに命令する際の2通りの方法について紹介する。

命令の記述

Gnuplotでは、好みや場合によって

  • コンソールに命令を1行ずつ記述、実行する方法(インタラクティブ)
  • ソースコードを書き、後でまとめて実行する方法(バッチ)
の2通りを選択することができる。

インタラクティブ

簡単な処理であれば、コンソールに直接入力したほうが早いので、こちらを推奨する。

Gnuplotをインストール済みのPCであれば、 

$ gnuplot
と入力することでGnuplotをCUIで起動できる。ここで1行ずつ 
$ plot sin(x)
と打ってみると、正弦関数のグラフが表示されるだろう。

なお、後述の.gpファイルを読み込むことができる： 

$ load "xxx.gp"

バッチ

複雑な処理の場合、複数の命令をまとめて記述・実行することが可能で(バッチ処理)、 通常は.gpファイルを用いる。文法はインタラクティブ的な方法と全く同じである。例えば： 

set xlabel "x"
set ylabel "y"
f(x) = sin(x)
g(x) = cos(x)
plot f(x) lw 2 t "sin", \
     g(x) lw 1 dt (3,3) t "cos"
などと入力し、scgraph.gpのような名前で保存すれば、このファイルを(gnuplotで)開く、 またはコンソールでloadすることで正弦関数と余弦関数のグラフを描画できるだろう。

Gnuplotでグラフを作る方法として、コンソールを用いるインタラクティブ的な方法と ソースコードを記述するバッチ処理的な方法の2種類がある。
