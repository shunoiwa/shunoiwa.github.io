1-2. 基本設定

このセクションでは、グラフを描画する際の基本的な設定について紹介する。

setコマンド

グラフ描画のための基本的な設定は、setコマンドによって行う： 

set option {suboption} {value}

ここでは最低限必要な設定についてのみ説明する。

terminal

Gnuplotでは、様々な実行環境が用意されている。これにより、出力形式が大幅に異なるため、 好みの環境を探してみよう。

terminalオプションでは環境を設定することができる。suboptionとして設定した環境を指定することができる。

set terminal env

とすると環境envに設定できる。このページでは、qt環境、wxt環境、またはpostscript環境のいずれかを 利用することを推奨する(もちろん、好みや用途に合わせて変えても良い)。

datafile

datafileオプションでは、ファイル読み込み(1-4)時の読み込み規則を指定する。

datafileのサブオプション一覧
suboption valueの種類 デフォルト値 説明 例
separator 文字列 半角空白文字 データ列の区切り文字を指定する set datafile separator ","
commentschars 文字列 "#"（VMSでは "#!"） コメント行の開始文字を指定する set datafile commentschars "#!%"
missing 文字列 / NaN 未設定 欠損値として扱う値を指定する set datafile missing NaN
columnheaders （なし） 無効 最初の行を列ヘッダとして扱う set datafile columnheaders
fortran （なし） 無効 Fortran形式（D, Q指数）を解釈可能にする（遅くなる） set datafile fortran
nofpe_trap （なし） 無効 浮動小数点例外チェックを省略して高速化（危険あり） set datafile nofpe_trap
binary binaryオプション列 未設定 バイナリデータとして読み込む設定 set datafile binary filetype=auto

その他のオプション

「その他」とは書いたが、実はここが一番大事である。

基本的なsetオプション一覧
option valueの種類 デフォルト値 説明 例
xlabel / ylabel 文字列 (なし) x軸・y軸のラベルを設定 set xlabel "Time"
title 文字列 (なし) グラフタイトルを設定 set title "Sample Plot"
xrange / yrange [min:max] 自動 表示範囲min~maxを指定 set xrange [0:10]
logscale x / y / xy (なし) 対数スケールにする set logscale y
grid （なし） 無効 グリッド線を表示 set grid
key 位置指定 / off 自動配置 凡例（legend）の表示・位置 set key left top
terminal 種類 環境依存 出力形式を指定 set terminal png
output 文字列 (なし) 出力先ファイルパスを指定 set output "graph.png"
style line (特定の形式なし) (なし) 線の色・太さなどを定義 set style line 1 lc rgb "blue" lw 2

なお、具体的な用法については第2章で紹介する。

unsetコマンド

unset option

で、設定した項目をデフォルト値に戻したり、無効化したりできる。

  • setコマンドにより、様々な設定や定義を行うことが出来る。
  • unsetコマンドにより、設定をデフォルト値または無効に設定できる。
